# Bluestock Fintech — Mutual Fund Analysis Capstone

An advanced data analytics, ETL pipeline, and portfolio risk management platform built to systematically ingest, process, and evaluate mutual fund schemes. This project analyzes historical NAV trends, tracks investor transaction cohorts, highlights churn vulnerabilities, and evaluates sector concentration metrics.

## 🛠️ Project Directory Layout
* `/data-raw`: Ingested raw datasets (fund master performance matrices, user transaction logs, exact portfolio allocations).
* `/data-processed`: Structured, deduplicated, and fully cleaned target data modules.
* `/notebooks`: Core analytical workbooks tracking exploratory data analysis (EDA) and complex statistical computations.
* `/reports/exported_charts`: High-resolution graphic assets generated by the analysis engines for management reporting.
* `run_pipeline.py`: Master automated orchestration script used to deploy and execute the backend environment cleanly.
* `recommender.py`: Command-line interactive asset advisory algorithm matching risk profiles with optimal Sharpe ratios.
